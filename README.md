# Login System

This is my first login and registration page project created using HTML, CSS, and JavaScript.

It includes a modern UI design with smooth animations and a toggle feature between login and registration forms.

✨ Features:
- Animated login & signup form
- Smooth transitions
- Responsive design
- Social login icons

🚀 Technologies Used:
- HTML
- CSS
- JavaScript

🔁 Toggle Feature:
When you click on the "Register" button, the registration form appears with a smooth animation.  
When you click on the "Login" button, it switches back to the login form.
  

This is my first self-made project and a step towards learning web development.
